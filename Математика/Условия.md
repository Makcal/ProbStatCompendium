Условные конструкции в математике можно записывать как формулы (functional programming moment). Для этого используются индикаторные функции:
$$f(x) = \begin{cases}
a\ \text{if cond1} \\
b\ \text{if cond2} \\
c\ \text{otherwise}
\end{cases} = 
aI(\text{cond1}) + bI(\text{cond2}) + cI(\overline{\text{cond1}}\ \&\ \overline{\text{cond2}})$$
(подразумевается, что условия не пересекаются между собой).
#### Пример
$$\xi=a\text{ if coin is heads, and }b\text{ if it is tails }$$
$$\xi = aI(heads) + bI(tails)$$
