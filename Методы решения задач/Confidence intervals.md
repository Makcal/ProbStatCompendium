Дан сэмпл и уровень уверенности, найти из данных сэмпла вероятные границы для неизвестного параметра.
# Алгоритм
1. Определить тип задачи.
2. Записать нужную формулу для границ интервала.
3. Найти границы по формуле, зная уровень доверия $\alpha$ (классическая задача), или найти уровень доверия/другой параметр, зная границы из условия, составив уравнение "граница по формуле = то что дано".
### Обозначения
- $\mu,\sigma^2$ — параметры распределения (не зависят от сэмпла)
- $S_n = \xi_1+...+\xi_n$ — сумма сэмпла
- $\bar x = \frac{S_n}{n}$ — sample mean, среднее арифметическое
- $s^2 = \frac{\sum(\xi_i-\bar x)^2}{n-1}, s=\sqrt{s^2}$ — sample variance, sample standard deviation 
- $V_n=\sum(\xi_i-\mu)^2$ — нечто
- $z_{\alpha}$ — такое число, которое надо найти по табличке **стандартного нормального**, чтобы $F_Z(z_\alpha)=\alpha$
- $w_{\alpha;n}$ — такое число, которое надо найти по табличке **чи-квадрат с параметром $n$**, чтобы $F_{\mathcal{X}^2_n}(w_{\alpha;n})=\alpha$
# Границы по кейсам
- $\xi_i \sim \mathcal{N}(\mu,\sigma^2)$, $\mu$ — оцениваем, $\sigma^2$ — известно:
	- $\mu \in \bar x \pm z_{1-\frac\alpha2}\frac{\sigma}{\sqrt{n}}$
- $\xi_i \sim \mathcal{N}(\mu,\sigma^2)$, $\mu$ — известно, $\sigma^2$ — оцениваем:
	- $\sigma^2 \in (\frac{V_n}{w_{1-\frac\alpha2}};\frac{V_n}{w_{\frac\alpha2}})$
	- Для $\sigma$ извлечь корень из границ отрезка выше.
- **При $n \ge 40$**: любое распределение, $\mu$ — оцениваем, $\sigma^2$ — неизвестно:
	- $\sigma \approx s$
	- $\mu \in \bar x \pm z_{1-\frac\alpha2}\frac{s}{\sqrt{n}}$
