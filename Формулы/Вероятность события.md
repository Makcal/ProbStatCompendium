## Вероятность <u>событий</u>
* [[Через элементарные исходы|Перебрать элементарные исходы]] события: $P(A)=P(\set{\omega_1,...,\omega_n})=P(\set{\omega_1})+...+P(\set{\omega_n})$
* Если элементарные исходы **равновероятны**:
    - Школьное/классическое/[[Комбинаторная вероятность|комбинаторное]] определение вероятности: $P(A)=\frac{\#desirable\_outcomes}{\#total\_outcomes}$
    - Если исходы можно представить, как точки с дробными координатами, и область подходящих/всех исходов можно *закрасить* на прямой/плоскости/пространстве, считаем по рисунку [[Графически|через площадь]]: $P(A)=\frac{area\_desirable}{area\_total\_possible}$
- **От противного** (стоит попробовать, может это выгодно): $P(A) = 1 - P(\bar A)$
- Вероятность **пересечения** $P(A\cap B) = P(A\ \&\ B) = P(AB)$. Начиная с самого удобного:
	- Для **независимых** событий: $P(A\ \&\ B)=P(A)\cdot P(B)$
	- Через условную вероятность (обычно применяется в пошаговом [[Двухфазный эксперимент|двухфазном эксперименте]] или в других случаях, когда можно найти условную вероятность): 
		- $P(A\ \&\ B)=P(B)\cdot P(A|B)=P(A)\cdot P(B|A)$
		- $P(A\ \&\ B\ \&\ C)=P(A)\cdot P(B|A)\cdot P(C|A\&B)$
		- и т.д.
	- **От противного**, если получится хорошее объединение: $P(A\ \&\ B) = 1- P(\bar A\cup \bar B)$
	- Любым другим **способом для события** $\set{A\ \&\ B}$, неспецифичным для пересечений.
- Вероятность **объединения** $P(A\cup B)$:
	- Для **непересекающихся** событий: $P(A\cup B) = P(A)+P(B)$
	- **От противного**, если получится хорошее пересечение: $P(A\cup B) = 1-P(\bar A\ \&\ \bar B)$
	- По формуле включений-исключений (когда полученные пересечения удобно считать): $P(A\cup B\cup C)=P(A)+P(B)+P(C)-P(AB)-P(BC)-P(AC)+P(ABC)$
- Формула полной вероятности (чтобы **рассматривать кейсы (варианты) по отдельности**):
	- $P(A\ \&\ B)=P(A\ \&\ B_1)+P(A\ \&\ B_2)+\ ...\ +P(A\ \&\ B_n)$
	- $P(B)=P(B_1)+P(B_2)+\ ...\ +P(B_n)$ (частный случай, когда $A=\Omega$, то есть просто рассматриваем кейсы одного события $B$)
	- $P(A)=P(A\ \&\ B_1)+P(A\ \&\ B_2)+\ ...\ +P(A\ \&\ B_n)$ (частный случай, когда $B=\Omega$, то есть рассматриваются **все возможные** кейсы события $A$; **самый часто используемый вариант**)
- Условная вероятность $P(A|B)$:
	- Формула Байеса (используем, когда известно что-то обратный порядок $P(B|A)$): $P(A|B)=\frac{P(B|A)P(A)}{P(B)}$
	  >Часто в формуле Байеса требуется применить **формулу полной вероятности** для знаменателя и найти пересечение **через условную вероятность**. Классический пример: тест положительный или если мы больны, или если мы не больны: $P(B) = P(B\ \&\ A) + P(B\ \&\ \bar A) = P(B|A)P(A) + P(B|\bar A)P(\bar A)$
	  
	- Определение: $P(A|B)=\frac{P(A\ \&\ B)}{P(B)}$
