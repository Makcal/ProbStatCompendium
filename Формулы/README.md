Формулы в этой папке показывают различные способы "**как что-то посчитать**". Ваша задача знать все способы и при необходимости посчитать нечто требуемое, **перебрать соответствующие варианты**. Некоторые формулы, такие как определения понятий, работают всегда, другие же — только при соблюдении условий, которые надо помнить.
