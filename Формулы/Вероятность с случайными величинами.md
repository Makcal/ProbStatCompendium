В этом файле рассматриваются вероятности вида $P(\xi=2), P(\xi\le6), P(\xi\in\set{1,2,3})$ и схожие. Здесь $\xi$ может быть любой случайной величиной (в теории, хоть $X_1 + X_2X_3$). 

Главный инструмент при ответе на вопросы про вероятность с.в. — это её **распределение**: pmf/pdf и cdf. 

>[!NOTE] Индикаторы
>Индикатор в pmf/pdf показывает **область значений** с ненулевой вероятностью. ПОЧТИ ВСЕГДА стоит подставлять pmf/pdf в формулы **вместе с индикатором**.

>[!info] Note
>Все условия внутри $P$ всё ещё являются **событиями**, поэтому к ним могут применяться все методы из [[Вероятность события]].
# Дискретные с.в.
- $P(\xi=x)=p_\xi(x)$ — вероятность конкретного значения
- **Самое главное**: $P(\xi\in\set{a,...,b}), P(a\le\xi\le b) = p_\xi(a)+...+p_\xi(b)$ — **перебрать все** подходящие значения и сложить их вероятности.
	- $P(\xi\le x), P(\xi> x)$ и т.п. — то же самое, но стоит проверить, может удобно пойти **от противного**, чтобы перебирать меньше значений
	- $P(\xi\le x)=F_\xi(x)=\sum_{x_0=-\infty}^{x_0=x}p_\xi(x_0)$ — cdf
	- $P(a\,\mathbf{<}\,\xi\le b) = P(\xi\in(a,b]) = F_\xi(b)-F_\xi(a)$ — редко используется с дискретными
### Вектор (несколько с.в.)
- $P(\xi=x,\eta=y)=p_{\xi,\eta}(x,y)$ — joint pmf
	- **Для независимых:** $p_{\xi,\eta}(x,y) = p_{\xi}(x) \cdot p_{\eta}(y)$
- $P(a\le\xi\le b,c\le\eta\le d)=p_{\xi,\eta}(a,c)+p_{\xi,\eta}(a+1,c)+...+p_{\xi,\eta}(b,d)$ — аналогично одной с.в.: перебрать **все** варианты значений
- **Marginal**: $p_\xi(x)=\sum_y p_{\xi,\eta}(x,y)$ — "исключить" (с помощью суммы) $y$ из joint pmf. По смыслу нам не важно значение $\eta$, поэтому раскладываем по формуле полной вероятности **все кейсы для** $\eta$ **и складываем их** как альтернативные варианты, получая в итоге событие "$\xi=x$, *любое значение для* $\eta$".
- Условная pmf: $P(\xi=x|\eta=y) = p_{\xi|\eta}(x|y) = \frac{p_{\xi,\eta}(x,y)}{p_{\eta}(y)}$
# Непрерывные с.в.
Здесь всё почти полностью аналогично дискретным, заменяя сумму на интеграл.
- $p_\xi(x)$ — pdf, **ключевой инструмент** всех формул.
- $P(\xi=x)=0$ — значений бесконечно много, поэтому у каждого по отдельности вероятность 0.
- **Самое главное**: $P(a\le\xi\le b) = \int_a^bp_\xi(x)dx$ — перебрать все подходящие значения и сложить их вероятности.
	- $P(\xi\le x)=F_\xi(x)=\int_{-\infty}^xp_\xi(x_0)dx_0$ — cdf
	- $P(a\le\xi\le b) = F_\xi(b)-F_\xi(a)$ — если известна cdf. Часто используется с табличками $\mathcal{N}(0,1), \mathcal{X}^2(n)$
### Вектор (несколько с.в.)
- $p_{\xi,\eta}(x,y)$ — joint pmf
	- **Для независимых:** $p_{\xi,\eta}(x,y) = p_{\xi}(x) \cdot p_{\eta}(y)$
- $P(a\le\xi\le b,c\le\eta\le d)=\int_a^b\int_c^d p_{\xi,\eta}(x,y)dydx$ — аналогично одной с.в.: перебрать **все** варианты значений. Границы и $d$ интегралов стоят симметрично: внешние к внешнему.
- **Marginal**: $p_\xi(x)=\int_{-\infty}^{+\infty} p_{\xi,\eta}(x,y)dy$ — "исключить" (с помощью интеграла) $y$ из joint pmf. По смыслу нам не важно значение $\eta$, поэтому раскладываем по формуле полной вероятности **все кейсы для** $\eta$ **и складываем их** как альтернативные варианты, получая в итоге событие "$\xi=x$, *любое значение для* $\eta$".
- Условная pmf: $P(\xi=x|\eta=y) = p_{\xi|\eta}(x|y) = \frac{p_{\xi,\eta}(x,y)}{p_{\eta}(y)}$
# Самое важное
- Дискретные: $P(a\le\xi\le b) = p_\xi(a)+...+p_\xi(b)$
- Непрерывные: $P(a \lt\xi\le b) = \int_a^bp_\xi(x)dx = F_\xi(b)-F_\xi(a)$
- Квантиль: такое число $x_{\alpha}$, что $F_\xi(x_{\alpha})=\alpha$
